# 📚 Open Library Book Finder

📌 **Project Description**
Open Library Book Finder is a beautifully designed, modern web application that allows users to search for books by title or author. It fetches real-time data from the Open Library API and beautifully displays book covers, titles, and publication years in a clean, responsive, glassmorphism-inspired interface.

## 🚀 Features
- 🔍 **Search**: Find books by title or author seamlessly.
- 📚 **Details**: Display book covers and key details effortlessly.
- 📅 **Publication Year**: Shows the first publication year.
- 📱 **Fully Responsive**: Optimized for mobile, tablet, and desktop views.
- 🎨 **Modern UI**: Implements a sleek dark mode with glassmorphism and smooth micro-interactions.

## 🌐 API Used
This project uses the purely open-source Open Library API.
[API Link](https://openlibrary.org/developers/api)

## 🛠️ Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Vanilla CSS, CSS Grid/Flexbox, Custom Properties, Animations)
- JavaScript (ES6+, Fetch API, Async/Await)

## ⚙️ Setup Instructions
1. Clone the repository or download the source code.
2. Open the project folder.
3. Open `index.html` in your favorite web browser (or use a local server like `live-server`).
